# 🧮 C# 電卓アプリ

C#（Windows Forms）で作成した **多機能電卓アプリ** です。
基本的な四則演算に加え、三角関数や平方根、対数などの**科学計算機能**も搭載しています。
また、テーマ変更や小数点以下の桁数設定も可能です。

---

## ✨ 主な機能

* 🔹 **基本演算**：加算、減算、乗算、除算
* 🔹 **科学関数**：

  * 平方根（√）、累乗（^）、逆数（1/x）
  * 三角関数（sin, cos, tan）※度数法／ラジアン切替対応
  * 常用対数（log₁₀）
* 🔹 **エラーハンドリング**：

  * 0 での割り算、log(0)、tan(90°) などを検出してエラー表示
* 🔹 **小数点以下の桁数設定**：1〜4 桁まで選択可能
* 🔹 **テーマ切替**：ライト／ダーク／ブルー
* 🔹 **簡潔で見やすいUI**（`Consolas` フォント使用）

---

## ⚙️ 実行方法

1. リポジトリをクローンまたはダウンロード


3. ビルドして実行（F5キー）
